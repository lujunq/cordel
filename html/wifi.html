WIFI SETTINGS


By deafult, the board starts as an access point named cordel with the password a9b8c7d6e5. After the first connection you may change this setting using the form below. You can also hold down the reset button for some time to return to the access point mode. After setting the wifi behaviour, the curcuit is automatically restarted. You can always check out the mode leds to see how the wifi is being handled by the server.
